TAT Breach Prediction & Alerts

A risk model scores every active order by comparing elapsed time + current stage against historical stage durations for that lens type. Orders crossing the risk threshold trigger an Email/WhatsApp alert to the assigned team before the SLA actually breaches.

Risk Model Summary

For each active order, predicted completion = elapsed time + average remaining-stage durations (from historical run-rate per lens type). If predicted completion exceeds the SLA deadline → Breach Alert. If predicted completion is within 80% of the SLA deadline → At-Risk Alert. Alerts are pushed to the store team via Email and WhatsApp simultaneously.
